Denpendencies
==================

GAMUT relies on following Python packages

+----------------+----------------+-------------------------------------------------+
| Package        | Version        | Functionality                                   |
+================+================+=================================================+
| numpy          | >1.25.0        | Used to realize matrix operation in algorithms  |
+----------------+----------------+-------------------------------------------------+
| pandas         | >2.1.4         | Used to import/export Spectrum objects          |
+----------------+----------------+-------------------------------------------------+
| matplotlib     | >3.7.2         | Used for spectrum visualization                 |
+----------------+----------------+-------------------------------------------------+
| scipy          | >1.11.1        | Used in some algorithms                         |
+----------------+----------------+-------------------------------------------------+
| pywt           | >1.4.1         | Used in some algorithms related to wavelet      |
|                |                | transformation                                  |
+----------------+----------------+-------------------------------------------------+
