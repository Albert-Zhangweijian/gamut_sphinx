gamut.classes.Peak module

class gamut.classes.Peak.Peak(location: int, center: float | None = None, height: float | None = None, stderr: float | None = None, **kwargs)[source]

Bases: object

Class representing a single peak in a spectrum. The only required parameter is the location/index of the peak. Parameters like center or height are optional, depending if the peak has been fitted. Other trivial parameters are stored in the attrs dictionary.

Attributes: - location: int, the index of the peak - center: Optinal[float], the center of the peak - height: Optinal[float], the height of the peak - stderr: Optinal[float], the standard error of the peak - attrs: dict, other trivial parameters that are not commonly used

Methods: - copy: return a deep copy of the peak - shift_by_index: shift the peak position by a given index
